# CARLA 运行指令与算法说明

本文包含：高性能 UE4 启动命令、写盘/不写盘运行指令、清理 UE4 命令，以及当前自动驾驶与多模态融合策略说明。

## 1) 高性能 UE4 启动

```powershell
D:\carla\CARLA_0.9.16\CarlaUE4.exe -carla-rpc-port=2000 -quality-level=Epic -vulkan -fps=120 -NoVSync -Windowed -ResX=1280 -ResY=720
```

## 2) 运行指令（写盘）

写盘会在 `C:\CARLA\carla_data\run_YYYYMMDD_HHMMSS` 下生成子目录和日志。

```powershell
python C:\CARLA\autonomous_driving_test.py --map Town03 --num-vehicles 6 --num-walkers 2 --sync --duration 300 --ego-adv --target-speed 55 --delta-time 0.02 --hud --vis-stride 5 --save-stride 5 --hazard-dist 12 --hazard-speed 18
```

## 3) 运行指令（不写盘）

仅实时显示，性能更稳。仍会显示 UE4 画面和实时多模态窗口。

```powershell
python C:\CARLA\autonomous_driving_test.py --map Town03 --num-vehicles 6 --num-walkers 2 --sync --duration 300 --ego-adv --target-speed 55 --delta-time 0.02 --hud --vis-stride 5 --no-save --hazard-dist 12 --hazard-speed 18
```

不写盘运行（更流畅）：

```powershell
python C:\CARLA\autonomous_driving_test.py --map Town03 --num-vehicles 6 --num-walkers 2 --sync --duration 300 --ego-adv --target-speed 55 --delta-time 0.02 --hud --vis-stride 5 --no-save --hazard-dist 12 --hazard-speed 18
```

## 4) 清理 UE4 进程

全部关闭（避免多实例冲突）：

```powershell
Get-Process | Where-Object { $_.ProcessName -like 'CarlaUE4*' } | Stop-Process -Force
```

## 5) 实时显示与多视角窗口说明

实时显示窗口标题为 `CARLA Multimodal Live View`，布局如下：

- 左上：RGB
- 右上：Depth
- 左下：Semantic
- 右下：状态面板（Radar/LiDAR 帧数、最近障碍距离、语义命中、红绿灯状态/距离）
- 底部 HUD（可选 `--hud`）：帧号、速度

同时默认会打开两个额外视角窗口（便于多屏展示）：

- `First Person`：第一人称视角
- `Third Person`：车尾第三人称视角

如需关闭额外视角窗口：

```powershell
--no-view-windows
```

## 6) 自动驾驶与多模态融合策略说明

### 6.1 车辆控制

- 采用纯跟踪式车道保持（pure-pursuit-like）进行转向。
- 纵向控制按目标速度分段给油/刹车，保证车辆保持在车道内并快速逼近目标速度。

### 6.2 多模态障碍融合

融合来源：

- LiDAR：前方走廊范围内点云距离（滤掉近场噪声）
- Radar：前方扇区内最小深度
- Semantic：前方 ROI 内车辆/行人标签比例

融合逻辑：

- 语义命中 + LiDAR/Radar 距离达到阈值才进入“确认状态”
- 确认计数达到阈值后开始降速
- 当距离进一步缩短时强制更低目标速度，避免追尾

### 6.3 红绿灯策略

使用 `--traffic-strategy` 控制行为：

- `stop`：红灯完全停车
- `slow`：红灯降速通过（测试模式）
- `detour`：忽略红灯

### 6.4 前车场景（可选）

使用 `--lead-vehicle` 在 Ego 同车道前方固定距离生成静态车，用于验证避障策略。

## Extra Options (ASCII)

- Move NPCs off ego lane (right/opposite lanes): `--npc-off-route`
- Fullscreen FP/TP windows for VR: `--view-fullscreen`
- Bigger FP/TP window size: `--view-width 1920 --view-height 1080`
